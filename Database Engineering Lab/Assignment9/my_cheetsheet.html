Display column with different name in result set.
Select From Multiple Tables
SELECT * FROM <table1>, <table2>;
Retrieve data from multiple tables (creates cartesian product).
Advanced SELECT Queries
Find Nth Largest Record
SELECT * FROM <table_name> ORDER BY <column> DESC LIMIT 1 OFFSET 1;
Find 2nd largest value (OFFSET 1 for 2nd, OFFSET 2 for 3rd, etc).
Example: Find 2nd highest salary
SELECT * FROM employees ORDER BY salary DESC LIMIT 1 OFFSET 1;
Find Nth Largest Using DENSE_RANK
SELECT * FROM (SELECT *, DENSE_RANK() OVER (ORDER BY <column> DESC) as rank FROM <table>) AS ranked WHERE rank = 2;
Find Nth largest value using window function (handles ties).
Example: Find 2nd highest salary with window function
SELECT * FROM (SELECT *, DENSE_RANK() OVER (ORDER BY salary DESC) as rank FROM employees) AS ranked WHERE rank = 2;
Find Nth Largest Using Subquery
SELECT MAX(<column>) FROM <table> WHERE <column> NOT IN (SELECT DISTINCT <column> FROM <table> ORDER BY <column> DESC LIMIT 1);
Find 2nd largest by excluding the 1st largest, then finding max of remaining.
Example: Find 2nd highest salary using subquery
SELECT MAX(salary) FROM employees WHERE salary NOT IN (SELECT MAX(salary) FROM employees);
Get Top 3 Records By Category
SELECT * FROM (SELECT *, ROW_NUMBER() OVER (PARTITION BY <category> ORDER BY <column> DESC) as rank FROM <table>) AS ranked WHERE rank <= 3;
Get top 3 records within each category using window function.
Example: Top 3 products by category ordered by price
SELECT * FROM (SELECT *, ROW_NUMBER() OVER (PARTITION BY category ORDER BY price DESC) as rank FROM products) AS ranked WHERE rank <= 3;
Concatenate Columns
SELECT CONCAT(<col1>, ' ', <col2>) AS full_name FROM <table>;
Combine multiple columns into single column.
INSERT - Data Insertion
Insert Single Row
INSERT INTO <table_name> (col1, col2) VALUES (val1, val2);
Insert one row with specified column values.
Insert Without Specifying Columns
INSERT INTO <table_name> VALUES (val1, val2, val3);
Insert row with values in same order as table columns.
Insert Multiple Rows
INSERT INTO <table_name> (col1, col2) VALUES (val1, val2), (val3, val4), (val5, val6);
Insert multiple rows in single statement.
Insert From Another Table
INSERT INTO <table1> (col1, col2) SELECT col1, col2 FROM <table2>;
Copy data from one table to another.
Insert Or Ignore Duplicates
INSERT IGNORE INTO <table_name> (col1, col2) VALUES (val1, val2);
Insert row but ignore if unique constraint violated.
Insert Or Update On Duplicate
INSERT INTO <table_name> (id, col1) VALUES (1, 'value') ON DUPLICATE KEY UPDATE col1='new_value';
Insert row or update if primary key already exists.
UPDATE - Data Modification
Update All Rows
UPDATE <table_name> SET <column> = <value>;
Update column value for all rows (use with caution).
Update With Condition
UPDATE <table_name> SET <col1> = <val1> WHERE <condition>;
Update rows matching WHERE condition.
Update Multiple Columns
UPDATE <table_name> SET col1 = val1, col2 = val2 WHERE <condition>;
Update multiple columns in same statement.
Update With Calculation
UPDATE <table_name> SET price = price * 1.1 WHERE category = 'electronics';
Update column by performing calculation on its current value.
Update From Another Table
UPDATE table1 t1 JOIN table2 t2 ON t1.id = t2.id SET t1.column = t2.column;
Update one table with values from another table.
Update With Limit
UPDATE <table_name> SET <column> = <value> LIMIT 10;
Update only first 10 matching rows.
DELETE - Data Removal
Delete All Rows
DELETE FROM <table_name>;
Delete all rows from table (use with caution).
Delete With Condition
DELETE FROM <table_name> WHERE <condition>;
Delete rows matching WHERE condition.
Delete With Limit
DELETE FROM <table_name> WHERE <condition> LIMIT 10;
Delete only first 10 matching rows.
Delete Using Subquery
DELETE FROM <table_name> WHERE id IN (SELECT id FROM <another_table>);
Delete rows based on result of subquery.
Delete From Multiple Tables
DELETE t1, t2 FROM table1 t1 JOIN table2 t2 ON t1.id = t2.id WHERE <condition>;
Delete rows from multiple tables in single statement.
Filtering And Sorting
WHERE Clause
Basic WHERE
SELECT * FROM <table_name> WHERE <column> = <value>;
Filter rows where column equals specific value.
WHERE With AND
SELECT * FROM <table_name> WHERE col1 = val1 AND col2 = val2;
Filter rows where both conditions are true.
WHERE With OR
SELECT * FROM <table_name> WHERE col1 = val1 OR col2 = val2;
Filter rows where either condition is true.
WHERE With NOT
SELECT * FROM <table_name> WHERE NOT <column> = <value>;
Filter rows where condition is false.
WHERE With IN
SELECT * FROM <table_name> WHERE <column> IN (val1, val2, val3);
Filter rows where column value matches any in list.
WHERE With NOT IN
SELECT * FROM <table_name> WHERE <column> NOT IN (val1, val2);
Filter rows where column value doesn't match any in list.
WHERE With BETWEEN
SELECT * FROM <table_name> WHERE <column> BETWEEN val1 AND val2;
Filter rows where column value is within range (inclusive).
WHERE With LIKE
SELECT * FROM <table_name> WHERE <column> LIKE 'pattern%';
Filter rows using pattern matching with LIKE operator.
WHERE With Wildcards - Starts With
SELECT * FROM users WHERE name LIKE 'A%';
Find names starting with letter A.
WHERE With Wildcards - Ends With
SELECT * FROM users WHERE name LIKE '%son';
Find names ending with 'son'.
WHERE With Wildcards - Contains
SELECT * FROM users WHERE name LIKE '%and%';
Find names containing 'and' anywhere.
WHERE IS NULL
SELECT * FROM <table_name> WHERE <column> IS NULL;
Filter rows where column value is NULL.
WHERE IS NOT NULL
SELECT * FROM <table_name> WHERE <column> IS NOT NULL;
Filter rows where column value is not NULL.
WHERE With Comparison Operators
SELECT * FROM products WHERE price > 100 AND stock < 50;
Filter using comparison operators (>, <, >=, <=, !=).
ORDER BY
Sort Ascending
SELECT * FROM <table_name> ORDER BY <column> ASC;
Sort results in ascending order (A to Z, 0 to 9).
Sort Descending
SELECT * FROM <table_name> ORDER BY <column> DESC;
Sort results in descending order (Z to A, 9 to 0).
Sort By Multiple Columns
SELECT * FROM <table_name> ORDER BY col1 ASC, col2 DESC;
Sort by first column ascending, then by second descending.
Sort With Condition
SELECT * FROM <table_name> WHERE <condition> ORDER BY <column> DESC;
Filter rows first, then sort results.
GROUP BY
Group By Single Column
SELECT <column>, COUNT(*) FROM <table_name> GROUP BY <column>;
Group rows by column and count rows in each group.
Group By Multiple Columns
SELECT col1, col2, COUNT(*) FROM <table_name> GROUP BY col1, col2;
Group rows by multiple columns and count.
Group With HAVING
SELECT <column>, COUNT(*) FROM <table_name> GROUP BY <column> HAVING COUNT(*) > 5;
Group rows and filter groups where count exceeds 5.
Group With WHERE And HAVING
SELECT category, AVG(price) FROM products WHERE stock > 0 GROUP BY category HAVING AVG(price) > 100;
Filter rows before grouping, then filter groups after aggregation.

JOINs - Combining Tables
INNER JOIN
SELECT * FROM table1 INNER JOIN table2 ON table1.id = table2.table1_id;
Return only matching rows from both tables.
LEFT JOIN
SELECT * FROM table1 LEFT JOIN table2 ON table1.id = table2.table1_id;
Return all rows from left table, matching rows from right table.
RIGHT JOIN
SELECT * FROM table1 RIGHT JOIN table2 ON table1.id = table2.table1_id;
Return all rows from right table, matching rows from left table.
FULL OUTER JOIN
SELECT * FROM table1 LEFT JOIN table2 ON table1.id = table2.id UNION SELECT * FROM table1 RIGHT JOIN table2 ON table1.id = table2.id;
Return all rows from both tables with matches where possible.
CROSS JOIN
SELECT * FROM table1 CROSS JOIN table2;
Cartesian product - combine every row from table1 with every row from table2.
SELF JOIN
SELECT a.name, b.name FROM employees a, employees b WHERE a.manager_id = b.id;
Join table to itself to compare rows within same table.
Multiple JOINs
SELECT * FROM table1 JOIN table2 ON table1.id = table2.t1_id JOIN table3 ON table2.id = table3.t2_id;
Join more than two tables in single query.
JOIN With WHERE
SELECT * FROM users u JOIN orders o ON u.id = o.user_id WHERE u.status = 'active';
Join tables and filter results using WHERE clause.

Backup And Restore (Command Line)
Backup Single Database
mysqldump -u username -p database_name > backup.sql
Export database to SQL file.
Backup All Databases
mysqldump -u username -p --all-databases > all_backup.sql
Export all databases to single file.
Backup Specific Tables
mysqldump -u username -p database_name table1 table2 > tables_backup.sql
Export specific tables only.
Backup Structure Only
mysqldump -u username -p --no-data database_name > structure.sql
Export only table structure without data.
Backup Data Only
mysqldump -u username -p --no-create-info database_name > data.sql
Export only data without table structure.
Restore Database
mysql -u username -p database_name < backup.sql
Import database from backup file.
